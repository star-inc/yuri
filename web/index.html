Yuri

Agentic Assistant

Yuri is

an Agentic Assistant.
It can execute tools and use search engines to complete tasks, possessing reasoning, planning, and tool-use capabilities.

Advantages

  • Capable of reasoning and planning to complete complex tasks
  • Can integrate with search engines and custom tools
  • Assists users in Discord servers with automated tasks

Challenges

  • Requires API keys and model configurations
  • May occasionally hallucinate or return unexpected tool output
